## Create a virtual environment :

### python 3.8.5 (Windows)

```
python -m venv env
```

### python 3.8.5 (MacOS)

```
python3 -m venv env
```

## Activate the virtual environment :

```
source env/bin/activate
```

## Installation:

```
pip install -r requirements.txt
```

(MacOS)

```
pip3 install -r requirements.txt
```
